This page uses the <link rel="layout" href="simple"> element to specify a custom layout using short name discovery.

How It Works

  1. The <link rel="layout" href="simple"> element tells Unify to look for a layout named "simple"
  2. Unify searches for _simple.layout.html in the _includes/ directory
  3. The layout provides the overall page structure with <slot name="..."> placeholders
  4. This page content fills the slots using <template data-target="..."> elements

Benefits

  • Explicit control: Choose exactly which layout to use per page
  • Short names: Use simple names instead of full file paths
  • Standard HTML: Uses familiar <link> element syntax
  • Fallback support: Layouts can include default content for slots
